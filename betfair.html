betfair

betfair is the primary function in the betfaiR package. It takes three arguments, your Betfair username, your password and your API key, if entered correctly then the function will return an environment with the various Betfair API methods (at the moment - 2015-09-10 - this is not all methods).

Usage

bf <- betfair(usr = username, pwd = password, key = api_key)
