clearedOrders

This function allows all cleared orders to be returned. You can ask for all closed orders to be returned by leaving the function empty or filter based on various parameters.

usage

bf$clearedOrders(betStatus = "SETTLED",
                 eventTypeIds = NULL,
                 eventIds = NULL,
                 marketIds = NULL,
                 runnerIds = NULL,
                 betIds = NULL,
                 side = "BACK",
                 from = NULL,
                 to = NULL)

arguments

param details
betStatus filter based on how the bet was settled, one of SETTLED, VOIDED, LAPSED, or CANCELLED
eventTypeIds retrieve data on orders per sport
eventIds retrieve data on orders per event
marketIds retrieve data on orders per market
runnerIds retrieve data on orders per runner
betIds retrieve data on specific bets
side retrieve data based on whether they were BACK or LAY
from date filter
to date filter, string should be in yyyy-mm-dd format

return

The function should return details about all closed orders via a list. This list might need a little bit of work in converting into a neater format.
